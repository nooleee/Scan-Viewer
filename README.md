# Scan Viewer 🏥

## 💡프로젝트 개요
Scan Viewer는 의료 분야에서 빠르고 정확한 진단과 의료진이 다양한 의료 영상을 빠르고 정확하게 판독할 수 있고 이를 효과적으로 처리하고 뷰잉 할 수 있는 웹 프로젝트입니다.

## ⭐ 주요 기능
### 🔒로그인 / 회원가입
- 로그인
- 회원가입
- 유효성 검사

### 🖼 워크 리스트
- 워크 리스트 검색
- 워크 리스트 정렬

### 🔍 DICOM 뷰어
- 뷰어 랜더링
- 뷰어 툴바(줌 인/아웃, 회전, 팬, 스택, ...)
  
### 📄 리포트
- 리포트 생성 및 수정
- 리포트 기록 남김
- 질병 코드 검색

### 💬 채팅
- 유저 간 채팅
- 실시간 업데이트
